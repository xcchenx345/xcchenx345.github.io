🔑 如何获取 API 密钥

M MiMo 密钥
1 访问 MiMo 开放平台，用小米账号登录
2 进入左侧「API 密钥」菜单，点击「创建密钥」
3 复制生成的 sk-... 密钥，粘贴到上方 MiMo 输入框
D DeepSeek 密钥
1 访问 DeepSeek 平台，注册/登录账号
2 进入「API Keys」页面，点击「创建 API Key」
3 复制密钥，粘贴到上方 DeepSeek 输入框

⚠️ MiMo API 已启动计费，请确保账户有足够余额。密钥仅保存在本地浏览器中，不会上传至任何服务器。
